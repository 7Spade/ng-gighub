# Guides / 操作指南

本目錄用於存放操作指南與教學文件。

## 目錄用途

此目錄用於存放各種開發、部署、測試相關的 how-to 指南和教學文件。

## 建議的文件類型

- **開發指南** - 開發流程、工具使用、最佳實踐
- **測試指南** - 單元測試、整合測試、E2E 測試指南
- **部署指南** - 部署流程、CI/CD 設定
- **問題排除** - 常見問題與解決方案
- **效能優化** - 效能調優技巧與工具
- **安全指南** - 安全最佳實踐與檢查清單

## 命名規範

使用 kebab-case 命名：
- `development-workflow.md`
- `testing-guide.md`
- `deployment-process.md`
- `troubleshooting.md`
- `performance-optimization.md`
